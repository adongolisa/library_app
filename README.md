# Library App

A basic book lending library application built with Ruby on Rails.

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/library_app.git
   cd library_app

2. Install Dependencies:
   
` bundle install`

3.Set up the database:
   ```bash
   rails db:create
   rails db:migrate

4. Run the application:
  `rails server`

Visit http://localhost:3000/books to view the app.
5. Run the test suite with
  `rails test`
